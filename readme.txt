梦境游戏是前几天一个QQ群中知道的.这几天正好在看Flash AS 2.0的书,

前几天本人发现的唱歌赚钱bug,一次200W..那个来的比买体育彩票还快.

该程序的作用是可以让你免费试穿游戏里所有的衣服.GM那件动态的除外.(本人该游戏曾用名 好大的动物)



程序附带源代码,VC6.0 Sp6 Windows 2003 Rc1 Sp2 下编译通过.
